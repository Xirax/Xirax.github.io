Typescript
Intsalowanie kompilatora Typescript npm install --save-dev typescript
Inicjowanie środowiska TS (Przygotowanie pliku konfiguracyjnego kompilatora - tsconfig.json) npx tsc --init
Instalowanie ts-node, który umożliwia współpracę z nodejs npm install --save-dev ts-node
Instalowanie wbudowanych typów dla node (w ten sam sposób możemy zainstalować typy do każdej paczki jeśli tylko ona takowe posiada) npm install --save-dev @types/node
